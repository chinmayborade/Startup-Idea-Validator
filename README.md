# Description :


You describe your idea — it runs 5 parallel AI analyses across market sizing, competition, feasibility, revenue, and risk — then synthesizes everything into one honest verdict: Go, Pivot, or Kill.


# Under the hood:

Parallel LLM chains for speed
ReAct agents doing real web + App Store research
Google Trends for market timing
Clean dark UI with animated results


Tech Stack Used - Langchain , Python , Groq API , FastApi
